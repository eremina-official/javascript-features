Deep and Shallow Copy

Deep copy - in the deep copy of object or array all values are copied by value.

Shallow copy - in the shallow copy of object or array some values are copied by reference (this way the values in the copied object still point to the values of the initial object, if these values are changed on the copied object, they are changed on the initial object as well).

Nested objects and arrays are always copied by reference. Ways to get them copied by value: to copy nested objest manualy or first stringify the object (JSON.stringify) and then parse the string into a new object (JSON.parse).
