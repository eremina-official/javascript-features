this

this is a function's keyword, a reference to a JavaScript contex object in which the current code is executing.

In usual functions value of this does not depend on where a function is lexically declared. Value of this is defined at a runtime depending on how the function is called.

Default binding of this in a plain function call: this is the global object (or undefined in the strict mode).

When function is called as an object's method: this is set to the object.

If the object's method is assigned to a new variable, and that variable is used as a reference to call the method, the value of this is set like if the function was called with the default binding (this will be global object or undefined).

In arrow functions this is bound lexically, that means in arrow function the value of this is set and bound to the enclosing lexical environment (means where the function is physically written).
