async await

The async function is a function which returns a Promise. The async function is created by writing the async keyword in front of it.

The await operator is used to wait for a value. This value can be a result of resolved Promise or any other value (which will be converted to a resolved Promise). The await pauses the async function execution until the Promise is fullfilled or rejected.
