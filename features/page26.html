Var, let and const

Var

  • Scope: if var is defined inside a function then it is limited to the function scope, it var is defined outside any function then it is in the global scope.
  • Can be reassigned and mutated within its scope.
  • Is hoisted to the top of its scope and initialised with value of undefined.

Let

  • Scope: block-scoped (the scope is limited to a block defined by curly braces).
  • Can be reassigned and mutated within its scope.
  • Is hoisted but not initialized with any value (if let is used before declaration it gives a Reference Error).

Const

  • Scope: block-scoped (the scope is limited to a block defined by curly braces).
  • Can not be reassigned, but if an object is defined with const then it can be mutated.
  • Is hoisted but not initialized with any value (if const is used before declaration it gives a Reference Error).
