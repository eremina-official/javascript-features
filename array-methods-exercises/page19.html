Array methods - find two items in an array which sum equals to a specific target number, sum and product of an array of integers, sum of individual index values for two arrays

Tasks:

  • getElementSum - function takes an array of numbers and a target number as arguments and returns items in the array (indices) that sum up to the provided target number.
  • sumAndProduct - function returns the sum and product of an array of integers.
  • sumIndices - function takes two arrays as arguments and computes the sum of each individual index value for the given arrays

Practical part: unit tested in page19.test.js.
