Array methods - isArray, turn array into a string, print elements of an array

Tasks:

  • isArray - function takes an input and defines whether the input is an array or not.
  • Array to string - function takes an array and a delimiter as arguments and joins all elements of the array inserting the delimiter between them.
  • Print elements - function takes an array of arrays as an argument and prints to the console all elements of the array.

Practical part: unit tested in page21.test.js.
