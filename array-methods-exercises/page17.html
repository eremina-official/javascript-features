Array methods - delete duplicates, union of two arrays, difference of two arrays

Tasks:

  • Delete duplicates - function takes an array as an argument and deletes all duplicate items in the array.
  • Union of two arrays - function takes two arrays as arguments and returns items which are present in both arrays.
  • Difference of two arrays - function takes two arrays as arguments and returns items which are unique for each array.

Practical part: unit tested in page17.test.js.
